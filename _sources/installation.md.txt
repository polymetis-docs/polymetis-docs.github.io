# Installation

1. Activate a conda environment with `python=3.8`
2. Clone repo

```bash
git clone git@github.com:fair-robotics/fair-robot-envs
cd fair-robot-envs
```
3. Install polymetis using conda
```bash
conda install -c file://$(eval pwd)/polymetis/conda/channel \
  -c fair-robotics \
  -c conda-forge \
  polymetis
```
