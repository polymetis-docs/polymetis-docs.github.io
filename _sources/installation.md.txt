# Installation

1. Activate a conda environment with `python=3.8`
2. Clone repo

```bash
git clone git@github.com:fair-robotics/fair-robot-envs
cd fair-robot-envs
```
3. Install fair-controller-manager using conda
```bash
conda install -c file://$(eval pwd)/fair_controller_manager/conda/channel \
  -c fair-robotics \
  -c conda-forge \
  fair-controller-manager
```
4. For simulation support, install fair-robot-envs
```bash
pip install -e fair_robot_envs
```
