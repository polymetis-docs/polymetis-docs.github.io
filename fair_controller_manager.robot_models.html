fair_controller_manager.robot_models.torch_rnea module¶

A differentiable PyTorch robot model implementing RNEA.

An old version of a PyTorch robot model. Too slow for real-time control at 1kHz, but still usable in simulation. A more updated version is located at https://github.com/facebookresearch/differentiable-robot-model.
